# Assignly

Assignly is a cross-platform mobile app built with React Native and Expo that helps students organize courses, track assignments, manage deadlines, and receive reminder notifications.

## Features
- Course creation and color coding
- Assignment creation with due date and time
- Dashboard for due today, upcoming, and overdue work
- Local reminder notifications
- Swipe-to-delete interactions
- Assignment detail view with submitted status
- Persistent local storage with AsyncStorage

## Tech Stack
- React Native
- Expo
- Expo Router
- TypeScript
- React Context
- AsyncStorage
- Expo Notifications
